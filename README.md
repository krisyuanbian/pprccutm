## Overview

This repository provides the code required to reproduce the numerical results presented in the poster. The implementation includes various tree ensemble models, TreeSHAP method, factor analysis, and principal component analysis.

---

## Repository Structure

### Code

- **`DeltaSOFA.ipynb`**  
  Jupyter notebook containing all scripts needed to reproduce the numerical analyses when **Delta SOFA** is treated as the outcome variable.

- **`LengthofStay.ipynb`**  
  Jupyter notebook containing all scripts needed to reproduce the numerical analyses when **Length of Stay** is treated as the outcome variable.

- **`FA_PCA.R`**  
  R script implementing **factor analysis (FA)** and **principal component analysis (PCA)** used in the study.
